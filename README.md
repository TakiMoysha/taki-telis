## Taki-Telis

It is a simple telegram bot, for my personal use.

- Send notifications on GitHub events such as a new commit.
- Using aiogram + DI (dishka).

### Env file

```.env
TELEGRAM_BOT_TOKEN
TELEGRAM_OWNER_ID
```
